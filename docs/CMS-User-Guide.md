# CMS User Guide

## Writer Workflow
1. Open writer studio at `/writer`.
2. Add title, slug, excerpt, and category.
3. Compose content in editor or use structured block-compatible format.
4. Add FAQ entries and SEO fields.
5. Save as draft or publish.

## Editorial Features
- Slug preview and SEO preview are visible in post payload behavior.
- Word count and reading time are computed automatically.
- Internal suggestion engine can be queried per post.
- Related articles are surfaced on article pages.
- Draft autosave persists remotely by `draft_key`.

## Publishing Rules
- Writers can draft and publish immediate posts.
- Scheduling future publication requires editor/admin capability.
- Sponsored flag and disclosure text are supported.
- Premium toggle is available for feature-gated rollout.

## Media
- Cover and inline media upload supported.
- YouTube embeds are validated and normalized.
- Ad slot data supports top/inline/sidebar structures.
